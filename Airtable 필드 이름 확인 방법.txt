Airtable 필드 이름 확인 방법 (422 에러 해결)
================================================================================

현재 에러: "Unknown field name: '생성형 AI점수'"

이 에러는 Airtable 테이블에 해당 필드가 없거나, 필드 이름이 정확히 일치하지 않을 때 발생해요.

================================================================================
1. Airtable에서 필드 이름 확인하기
================================================================================

1.1. Airtable 웹사이트 접속
- https://airtable.com 에 로그인하세요

1.2. Base 열기
- "AX 역량모델" Base를 클릭해서 엽니다

1.3. 테이블 열기
- 왼쪽 사이드바에서 "사용자_평가결과" 테이블을 클릭합니다

1.4. 필드 이름 확인
- 테이블 상단의 필드 헤더(컬럼 이름)를 확인하세요
- 각 필드 이름을 정확히 확인하세요:
  * 공백이 있는지 확인
  * 대소문자가 다른지 확인
  * 특수문자가 다른지 확인

3. 필드 이름 수정 방법
================================================================================

3.1. 방법 1: Airtable 필드 이름을 코드와 일치시키기 (추천)

1. Airtable에서 "사용자_평가결과" 테이블을 엽니다
2. 필드 헤더를 클릭합니다 (또는 필드 이름 옆의 드롭다운 메뉴 클릭)
3. "Rename field" 또는 "필드 이름 변경"을 선택합니다
4. 필드 이름을 코드와 일치하도록 변경합니다:
   - 예: "생성형 AI점수" → "생성형AI점수" (공백 제거)
   - 또는 코드에서 사용하는 이름으로 변경

3.2. 방법 2: 코드의 필드 이름을 Airtable과 일치시키기

1. Airtable에서 실제 필드 이름을 확인합니다
2. js/airtable-api.js 파일을 엽니다
3. 필드 이름을 Airtable과 일치하도록 수정합니다

================================================================================
4. 빠른 확인 방법
================================================================================

4.1. Airtable에서 필드 이름 복사하기

1. Airtable에서 필드 헤더를 클릭합니다
2. 필드 이름을 복사합니다 (Ctrl+C)
3. 코드의 필드 이름과 비교합니다
4. 정확히 일치하는지 확인합니다

4.2. 필드 이름이 다른 경우

- Airtable 필드 이름: "생성형 AI점수" (공백 있음)
- 코드 필드 이름: "생성형AI점수" (공백 없음)
→ 둘 중 하나를 수정해야 해요!

================================================================================
5. 디버깅 팁
================================================================================

5.1. 콘솔에서 확인

브라우저 개발자 도구(F12) → 콘솔 탭에서:
- "📋 저장할 필드 이름들:" 메시지를 확인하세요
- 실제로 전송되는 필드 이름 목록이 표시됩니다

5.2. Network 탭에서 확인

브라우저 개발자 도구(F12) → Network 탭:
- "airtable.com"으로 시작하는 요청을 찾으세요
- 실패한 요청(422 에러)을 클릭하세요
- "Payload" 탭에서 전송된 필드 이름을 확인하세요
- "Response" 탭에서 에러 메시지를 확인하세요

6. 체크리스트
================================================================================

□ 1. Airtable에서 "사용자_평가결과" 테이블 열기
□ 2. 각 필드 헤더를 클릭해서 정확한 이름 확인
□ 3. "생성형 AI점수" 필드의 정확한 이름 확인 (공백 여부)
□ 4. 다른 필드들도 모두 확인
□ 5. 필드 이름이 코드와 일치하는지 확인
□ 6. 불일치하는 필드 이름 수정 (Airtable 또는 코드)
□ 7. 브라우저 완전 새로고침 (Ctrl + Shift + R)
□ 8. 설문 다시 완료
□ 9. 콘솔에서 에러 확인

7. 예상되는 필드 이름 변형

- "생성형AI점수" (공백 없음)
- "생성형 AI점수" (공백 있음)

- "GenAI점수" (영문)
- 다른 이름
