================================================================================
Airtable "Create base" 버튼 찾기 가이드
================================================================================

Base 이름을 입력한 후 "Create base" 버튼을 찾을 수 없을 때의 해결 방법입니다.

================================================================================
1. "Create base" 버튼 위치
================================================================================

1.1. 일반적인 위치
Base 이름을 입력하는 화면에서:

┌─────────────────────────────────────────┐
│  Create a new base                       │
├─────────────────────────────────────────┤
│                                          │
│  Name your base:                        │
│  [AX 역량모델________________]          │
│                                          │
│  [Cancel]  [Create base]  ← 여기!      │
│            또는                          │
│            [Create]                     │
└─────────────────────────────────────────┘

버튼은 보통:
- 입력창 아래쪽에 있습니다
- 오른쪽 하단에 있습니다
- "Cancel" 버튼 옆에 있습니다

1.2. 버튼이 보이지 않는 경우

방법 1: 스크롤 확인
- 화면을 아래로 스크롤해보세요
- 버튼이 화면 밖에 있을 수 있어요

방법 2: 입력창 확인
- Base 이름을 입력했는지 확인하세요
- 일부 버튼은 이름을 입력해야 나타날 수 있어요

방법 3: Enter 키 사용
- Base 이름을 입력한 후
- Enter 키를 눌러보세요
- Enter 키로도 Base를 생성할 수 있어요

방법 4: 화면 크기 조정
- 브라우저 창 크기를 조정해보세요
- 작은 화면에서는 버튼이 숨겨질 수 있어요

================================================================================
2. Base가 이미 생성된 경우
================================================================================

2.1. 확인 방법
화면 상단에 "AX 역량모델"이라는 제목이 보이고,
"Table 1"이라는 테이블이 보인다면
→ Base가 이미 생성된 상태입니다!

2.2. 다음 단계
이 경우 "Create base" 버튼은 필요 없어요.
다음 단계로 진행하세요:

1. 테이블 이름 변경
   - 왼쪽 상단의 "Table 1"을 클릭하세요
   - 이름을 "사용자_평가결과"로 변경하세요

2. 필드 추가
   - 테이블 설정 가이드에 따라 필드를 추가하세요

================================================================================
3. 문제 해결
================================================================================

3.1. 버튼이 여전히 보이지 않아요

해결 방법:
1. 페이지를 새로고침하세요 (F5)
2. 브라우저를 완전히 닫고 다시 열어보세요
3. 다른 브라우저로 시도해보세요 (Chrome, Edge 등)
4. 브라우저 확장 프로그램을 비활성화해보세요

3.2. 버튼을 클릭했는데 반응이 없어요

해결 방법:
1. 인터넷 연결을 확인하세요
2. 잠시 기다려보세요 (서버 응답 시간)
3. 브라우저 콘솔(F12)에서 에러 메시지를 확인하세요

3.3. "Create base" 대신 다른 버튼이 보여요

가능한 버튼 이름:
- "Create"
- "Create base"
- "완료"
- "만들기"
- "✓" (체크 아이콘)

모두 같은 기능이에요!

================================================================================
4. 단계별 체크리스트
================================================================================

□ 1. Airtable 홈 화면에서 "+ Add a base" 버튼 클릭
□ 2. Base 이름 입력창에 "AX 역량모델" 입력
□ 3. 입력창 아래 또는 오른쪽 하단에서 "Create base" 버튼 찾기
□ 4. 버튼이 보이지 않으면 Enter 키 눌러보기
□ 5. Base가 생성되면 "AX 역량모델" 제목이 상단에 표시됨
□ 6. "Table 1" 테이블이 보이면 성공!

================================================================================

================================================================================

================================================================================
6. 빠른 해결 방법
================================================================================

가장 빠른 방법:
1. Base 이름 "AX 역량모델"을 입력하세요
2. Enter 키를 누르세요
3. Base가 생성됩니다!

또는:
1. Base 이름을 입력하세요
2. 마우스를 입력창 아래로 이동하세요
3. "Create base" 또는 "Create" 버튼을 찾아 클릭하세요
