================================================================================
Airtable 필드 이름 확인 및 수정 가이드
================================================================================

422 에러 "Unknown field name"은 필드 이름이 Airtable과 일치하지 않을 때 발생해요.
에러 메시지: "Unknown field name: '생성형 AI점수'"

================================================================================
1. 문제 원인
================================================================================

현재 상황:
- 코드에서 사용하는 필드 이름: '생성형AI점수' (공백 없음)
- Airtable이 찾는 필드 이름: '생성형 AI점수' (공백 있음) 또는 다른 이름

필드 이름이 정확히 일치하지 않으면 422 에러가 발생해요!

================================================================================
2. 해결 방법
================================================================================

2.1. Airtable에서 실제 필드 이름 확인

1. Airtable 웹사이트를 엽니다
2. "AX 역량모델" Base를 엽니다
3. "사용자_평가결과" 테이블을 엽니다
4. 각 필드의 제목(헤더)을 정확히 확인하세요:
   - 공백이 있는지 확인
   - 대소문자가 다른지 확인
   - 특수문자가 다른지 확인

2.2. 필드 이름 수정 방법

방법 1: Airtable 필드 이름을 코드와 일치시키기 (추천)
1. Airtable에서 필드 이름을 클릭하세요
2. 필드 이름을 코드와 일치하도록 변경하세요
   예: "생성형 AI점수" → "생성형AI점수"

방법 2: 코드의 필드 이름을 Airtable과 일치시키기
1. Airtable에서 실제 필드 이름을 확인하세요
2. 코드의 필드 이름을 Airtable과 일치하도록 수정하세요

================================================================================
4. 필드 이름 수정 방법 (코드 수정)
================================================================================

4.1. Airtable에서 실제 필드 이름 확인 후 코드 수정

예시: Airtable 필드 이름이 "생성형 AI점수" (공백 있음)인 경우

js/airtable-api.js 파일에서:
'생성형AI점수' → '생성형 AI점수'로 변경

4.2. 모든 필드 이름 확인

Airtable에서 각 필드의 정확한 이름을 확인한 후,
js/airtable-api.js 파일의 모든 필드 이름을 일치시키세요.

================================================================================
5. 빠른 해결 방법
================================================================================

가장 빠른 방법: Airtable 필드 이름을 코드와 일치시키기

1. Airtable에서 "사용자_평가결과" 테이블을 엽니다
2. 각 필드 이름을 클릭해서 확인합니다
3. 필드 이름을 코드와 일치하도록 변경합니다:
   - "생성형 AI점수" → "생성형AI점수" (공백 제거)
   - 다른 필드도 동일하게 확인

또는

코드의 필드 이름을 Airtable과 일치시키기:
1. Airtable에서 실제 필드 이름을 확인합니다
2. js/airtable-api.js 파일을 엽니다
3. 필드 이름을 Airtable과 일치하도록 수정합니다

================================================================================
6. 필드 이름 확인 체크리스트
================================================================================

□ 1. Airtable에서 "사용자_평가결과" 테이블 열기
□ 2. 각 필드 헤더를 클릭해서 정확한 이름 확인
□ 3. 공백, 대소문자, 특수문자 확인
□ 4. 코드의 필드 이름과 비교
□ 5. 불일치하는 필드 이름 수정 (Airtable 또는 코드)
□ 6. 브라우저 새로고침 (Ctrl + Shift + R)
□ 7. 설문 다시 완료
□ 8. 콘솔에서 에러 확인

================================================================================
7. 에러 메시지 해석
================================================================================

에러: "Unknown field name: '생성형 AI점수'"

이 에러는:
- Airtable에 '생성형 AI점수'라는 필드가 없다는 의미
- 또는 필드 이름이 정확히 일치하지 않는다는 의미

해결:
- Airtable에서 필드 이름을 정확히 확인
- 코드의 필드 이름과 일치시키기
